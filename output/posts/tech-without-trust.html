Tech without trust

2026-07-06

A lot of software engineers like to solve social problems with technical solutions. Football is the proof that not everything can be solved with more technology. The ball of this year's world cup has a chip which can tell if someone touched it. Which feels like a good tool, if the ball touches the hand of a player, we don't need to rely only on the cameras (VAR), but also have more data to come to a conclusion. The problem is that data without transparency is useless.

During a match we have at least three distinct data sources: the referees in the field; the VAR and now the chip. When reviewing a match moment, the VAR needs to synchronise the chip data with the camera data, to make sure the data match the right moment. This is not as simple as it looks like and there is no formal document stating how they are doing it. Another point is that we cannot audit the data, FIFA does not release the chip data in a public space so that it can be validated. Data here is used only as a means to shut up critics. It's almost like that co-worker that throws numbers at you during design decisions without ever providing their source or a way to verify it. "Trust me, bro".

There is a moment on Croatia vs Portugal, where if the ball touched the head of the Croatia player, the player that scored was offside. If you watched the game, you cannot see the touch from any angle. But VAR with the usage of the chip declares that there was a touch. The TV shows a replay with only one chart, only for that moment (so we cannot really see if the data is synchronised correctly). You need to believe that even tho your eyes cannot see, the ball touched the head of the player.

If you say: "I cannot see the touch, the chip data is just used as a decoy", you are crazy and anti-tech. You are the same as the anti-vaxx people. The difference, tho, is huge. With vaccines we have the studies, the data, we can validate ourselves if needed. Any lab can prove the other studies wrong if so. With the world cup solution, we have the tech, but we cannot validate it. You have to trust.

FIFA is clearly not an entity that can be trusted, as it lifted a suspension from a USA player without any valid reason (well, FIFA also never banned Israel from football even tho it did with Russia and gave a "peace prize" to Trump). So we cannot trust the data from FIFA. It's not a conspiracy theory, if they want us to believe the data they show us, let us validate it.

This is not the only place where we are living a "trust us, bro" moment. Tech with AI is exactly that. We have some companies from the USA saying AI is way too dangerous to be on the wrong hands and we need to trust them (nationals from a country known for its war crimes) to be the guardians of it.

"Trust me, bro".
